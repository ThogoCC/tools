## ./code_helper/tabs2space.py

convert one tab character to four space character

@usage:

```bash
    ./tab2space.py --file file1 file2
```
```bash
    ./tab2space.py --dir directory  
```
```bash
    ./tab2space.py --file file1 file2 --keepname 1  
```
